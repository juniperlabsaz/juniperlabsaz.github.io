INTERNAL DEVELOPER TOOL

Automated social posting for indie mobile games

JuniperLabs Auto Post is a single-tenant internal automation tool that publishes short-form gameplay clips of our indie mobile games to our own social media creator accounts on a scheduled basis.

Overview

JuniperLabs is an independent mobile game studio with three published games on the App Store. JuniperLabs Auto Post is the internal tool we built to promote those games on TikTok, YouTube Shorts, and X by automatically publishing short gameplay clips with AI-generated captions.

LetterPath Word puzzle game
Yuki Grid-based puzzle game
Cow Dog Life Endless runner

What the tool does

The tool runs locally on JuniperLabs hardware. It does not collect any data from TikTok end users and is not made available to any third party.

📹

Gameplay capture

Selects pre-recorded gameplay clips from a curated local library of our own games.

🤖

AI caption generation

Uses Google Gemini to generate platform-tailored captions, hashtags, and hook text from a representative frame.

🎬

Vertical video build

Builds 9:16 vertical videos with branded text overlays using FFmpeg.

📤

Scheduled publishing

Publishes to JuniperLabs-owned creator accounts on a cron schedule via the official platform APIs.

How it works

A scheduled job runs three times per day per platform. Each run completes the following pipeline end-to-end.

1
Authentication JuniperLabs creator account is authorized once via TikTok's OAuth 2.0 flow with the user.info.basic and video.upload scopes. The access token is stored locally and refreshed automatically.
2
Clip selection The tool picks a gameplay clip from the local clip library, cycling through all clips before repeating to ensure variety.
3
Caption generation A frame is sent to Google Gemini, which returns a caption, hashtags, and a short hook overlay.
4
Video assembly FFmpeg builds a 9:16 vertical video with the hook overlay, music, and the gameplay footage.
5
Publish to TikTok The video is uploaded to the authorized JuniperLabs creator account via the TikTok Content Posting API. The post ID is logged locally.

TikTok API usage

We request the minimum scopes required for the integration. We do not collect data from any TikTok user other than the JuniperLabs creator account that explicitly authorizes the tool.

user.info.basic

Read the open ID, union ID, avatar, and display name of the JuniperLabs creator account to confirm authorization and label posts in our internal log.

video.upload

Upload pre-recorded gameplay videos to the authorized JuniperLabs creator account via the Content Posting API.
